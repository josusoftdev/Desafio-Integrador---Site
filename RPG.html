    A seguinte loja virtual foi criada como um projeto de programação para o curso de Engenharia de Software do Centro Universitário Campo Real (CUCR). O objetivo é demonstrar habilidades em desenvolvimento web, incluindo HTML, CSS e JavaScript. Não há qualuqer fim comercial ou venda de produtos reais. Todos os produtos e imagens são de propriedade de seus respectivos donos e foram utilizados apenas para fins educacionais. A loja virtual é uma simulação e não representa uma plataforma de comércio eletrônico real. Agradecemos a compreensão e esperamos que este projeto sirva como um exemplo de aplicação prática das habilidades adquiridas no curso.
